# iso_utilities

A collection of Python tools for working with the [ISOTROPY](https://iso.byu.edu/isolinux.php) software suite.

## Tools

### invariants
Generates invariant polynomials from ISOTROPY with physically meaningful variable names,
time-reversal symmetry filtering, and multiple output formats.

See `invariants/README.md` for usage.

## Requirements
Each tool has its own `requirements.txt`. ISOTROPY must be installed separately and
the `iso` executable must be on your PATH (or its location passed explicitly).
